# 어려웠던 부분
> 직접 사용했던 코드가 아니라 모듈을 불러오고 학습 자료를 통해서 직접 코드를 구현해야 했기에 생성형 AI를 최대한 사용하지 않고 결과를 도출해내기 어려웠습니다. 
> 내용이 크게 많지는 않아 같은 팀원으로 배정되었던 임지영님과 같이 같은 문제를 풀면서 서로 다른 코드를 구현해보고 이를 설명하는 시간을 가지면서 의견을 공유하였습니다. 아무래도 문제를 분담하여 과제를 수행하였으면 더 일찍 끝났겠지만, 팀원끼리 서로 많은 문제를 풀어보면서 배워보는 시간을 가지는 것이 어렵긴 했지만, 뜻 깊은 시간이었던 것 같습니다.

# 느낀 점
> 문제의 양이 많지 않고 학습 자료와 생성형 AI를 적절히 이용하여 코드를 구현하면 쉽게 끝날 것이라고 생각하였지만, 생각한 대로 잘 되지 않고 예상지 못한 곳에서 오류가 발생하여 시간을 많이 소요하기도 하였습니다. 항상 AI를 활용할 때에도 코드가 어떤 용도로 활용되는 것인지 직접 생각해보고 분석하는 것의 중요성을 알게 되었습니다.
