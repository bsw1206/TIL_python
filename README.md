## 🚀 Featured Learning: Data Engineering Roadmap
현재 파이썬을 기반으로 구축 중인 실시간 데이터 파이프라인의 핵심 개념을 정리하고 있습니다.

- **Ingestion:** Python Kafka Producer를 활용한 가상 센서 데이터 생성
- **Processing:** Spark Streaming을 활용한 실시간 윈도우 연산 및 데이터 정제
- **Storage:** 처리된 데이터를 S3 데이터 레이크 및 DB에 안정적으로 적재하는 법
- **Monitoring:** Prometheus & Grafana를 통한 시스템 상태 관측
