# CNN 원리 이해, CNN 연산, Pooling, 스트라이드 합성곱


# CNN vs FCN
- FCN(완전 연결층) : 입력을 받아서 출력으로 변환하는 신경망의 기본 모듈(1차원 구조로 변환)
- CNN(합성곱 레이어) : 입력 이미지를 필터와 연산하여(필터를 이동시킴, 내적을 반복 수행) 특징 맵을 뽑아내는 모듈(3차원 구조를 그대로 보존)
- wx + b(w : 필터가 덮는 이미지 영역, x : 필터의 가중치, b : 편향(보정값))
- 하나의 이미지에 여러 개의 필터를 적용하게 되면 필터의 개수 만큼 여러 개의 채널을 결과물로 얻는다.
- 출력 해상도? 입력 해상도-필터해상도 + 1
- 필터는 보통 홀수 * 홀수다.(중앙값이 존재하기 때문에)
- 패딩 : 출력값 중 정의되지 않은 경우, 0 혹은 가장 가까운 출력값으로 대체


# CNN 핵심 구조
- 합성곱 레이어, 풀링, 비선형 함수, 정규화, 연결층 레이어


# 풀링 
- 학습 상수 X
- 효율적 연산 및 위치 변화의 강건성 확보
- CNN 레이어의 출력을 줄여 연산 효율성을 확보: 입력 크기를 감소시킴, 2*2 인 경우 반으로 감소
- 맥스풀링 : 정해진 커널 사이즈로 이미지를 나누어 각 영역 내 가장 큰 값을 선택하는 연산
- 스트라이드 합성곱 : 풀링 한계 계선(필터를 S칸씩 이동한 후 출력 연산. <- 일반적인 필터를 1칸씩 이동의 한계 극복) / 학습 상수 존재

# CNN 모델 변천사
1. AlexNet : 5개의 합성곱 계층과 3개의 완전연결 계층으로 구성된 8계층 CNN 모델(8-레이어)
- GPU 기반 대규모 CNN의 첫 성공, ReLU 도입, Dropout을 통한 과적합 억제
2. VGGNet : 5개의 합성곱 블록 + 맥스풀링 구조, 3개 연결층 레이어, ReLU 비선형 활성화, 16개의 합성곱/연결층 레이어
- 3*3conv 반복
3. GoogLeNet : inception 모듈을 도입하여 다양한 크기의 필터를 병렬로 적용
4. ResNet : 합성곱 블록과 잔차 블록
- 등장 배경 -> 배치 정규화 방식으로 10+ 레이어 학습 가능 / 작은 모델의 성능은 최소한 누릴 수 있도록 추정값을 그대로 후속 레이어에 제공 
- 입력 X가 두 경로로 나뉘어 전달됨. / 마지막에 두 값 더하기
5. MobileNet : 모바일, 임베디드 환경에서 구동 가능 / 공간과 채널을 두 단계로 분리하여 처리 / 파라미터, 연산량 대폭 감소

# CNN 구조의 장점
- 지역적인 특징 학습에 특화, 파라미터 효율성 우수, 위치 변화와 노이즈에 견고, 이미지 기반의 업무에서 표준 모델로 활약

# CNN 한계
- 데이터를 무시하기 때문에 순차적 데이터의 순서를 반영하기 어려움.
- 멀리 떨어진 요소 간의 관계를 잘 학습하지 못함.
- 픽셀 단위 복원 한계

# RNN
- 순차적 데이터를 처리하기 위해 고안된 신경망 구조(이전 단계 데이터를 은닉 상태로 다음 단계에 전달)
- 시계열 데이터 처리에 관한 강력한 귀납 편향 제공
- RNN 셀 : 은닉층에서 활성화 함수를 통해 결과를 내보냄.
- 단순 RNN 한계 : 기울기 폭발 혹은 소실 발생 / 강제 절삭! 그러나 결국 잘못된 기울기 값을 양산하는 문제

# RNN 대안 모델
- LSTM : 셀 상태, 게이트 유닛으로 구성
- 셀 상태 : 중요한 정보를 유지/기억
- 게이트 : 정보의 흐름을 제어(입력 게이트, 망각 게이트, 출력 게이트)
- 구동 철학 : 긴 데이터에서 필요한 정보는 유지, 불필요한 정보는 지움.

### 어탠션 매커니즘
- 유사한 패치 정보를 최종 결정에 반영

# 교차 어탠션
- 둘 이상의 이종 데이터에서 패치 정보(쿼리,키,값) 정의 / 이종 데이터 간 유사도 반영

# 모델 별 특성 정리
- CNN : 지역적 특성에 집중 / 장거리 맥락 이해 떨어짐 / 데이터 순서 반영 불가
- RNN : 데이터 순서를 반영 / 장거리 맥락 학습이 어려움 / 기울기 소실 / 병렬 연산 어려움
- LSTM : 데이터 순서 반영 / 장거리 맥락 이해 가능 / 정보 희석 문제
- Attention : 장거리 맥락에 탁월 / 정보 희석 없음 / 데이터 순서?

# ViT vs ResNet
- ViT가 반드시 CNN보다 좋은 것은 아님.
- ViT는 거대 데이터세트 사전학습시 우수


# 활성화 함수
- 신경망에 비선형성 부여하여 복잡한 패턴 학습 가능
- 대표적인 예시 : 시그모이드 함수, tanh(범위 : -1에서 1까지), ReLU -> f(x) = max(0,x)(입력이 음수면 0만 출력), Leaky ReLU -> f(x) = max(0.01x, x)

# 모델별 전처리 방식 예시
- AlexNet : 평균 이미지를 빼기
- VGG : 채널별 평균 빼기
- ResNet : 평균 빼기 + 채널별 표준편차로 나누기

# L1 정규화, L2 정규화
- L1 정규화(Lasso) : 모든 가중치에 절댓값 크기만큼 동일하게 패널티
- L2 정규화(Ridge) : 큰 가중치에 제곱으로 패널티 -> 비정상적으로 큰 값 방지
- Elasticnet(L1 + L2)

# Dropout
- 학습 과정에서 일부 뉴런을 확률적으로 끔.
- 과적합 방지

# 학습 안정성 전략
- 학습 비율은 가장 중요한 하이퍼파라미터!
- 학습 비율은 큰 값에서 시작하여, epoch 지날수록 작은 값으로 조정(epoch : 학습 데이터를 한번 다 외움.)
- 직선에 따른 변경 코사인에 따른 변경 역제곱에 따른 변경 일정값 유지
- 빠른 종료 기법(과적합 방지) : 성능이 나빠지기 시작하면 학습을 조기에 멈춤.

# 하이퍼파라미터 선정
- 파라미터 : 모델 스스로 얻는 값 / 하이퍼파라미터 : 학습 전에 사용자가 정해야 하는 값
- 하이퍼파라미터 값 탐색 방식 : 그리드 탐색, 랜덤 탐색

# 체크리스트 
- 초기 손실값 확인 -> 학습률과 초기값 확인 -> 2에서 얻은 구조, 최적화 툴 활용, 학습률을 찾기 -> 3에서 얻은 학습률 + 후보 가중치 변형 방식 중, 좋은 조합 탐색 -> 4에서 얻은 좋은 조합으로 10~20에폭까지 추가학습 -> 학습률에 따른 손실곡선 관찰
