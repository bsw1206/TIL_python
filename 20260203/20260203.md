# 카운팅 정렬
- 집합에 각 항목이 몇 개씩 있는지 세는 작업을 진행함.
- 정수나 정수로 표현할 수 있는 자료에 대해서만 적용됨.
- 누적 합을 구하는 과정
- 누적 합이 구해지면 여러 작업을 빠르게 진행 가능함.
- 거꾸로 뒤집어서 누적합에서 하나를 뺌.
- 0으로 채워져 있는 TEMP리스트에다가 count[data[i]]위치에 count
- 특정 숫자에 누적합을 확인하고, 나열을 진행할 때 특정 숫자를 확인한 순간 최소 1개는 있다고 판단하고 맨 뒷자리에 고정을 시키면서 오름차순 정렬을 진행.
